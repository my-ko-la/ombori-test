# Ombori Code Test

## Running the project

`npm i`

`npm run start`

## Stack Used

- UI: TailwindCSS
- Data Fetching: TanStack Query
- Misc: react-intersection-observer
- Linter: ESLint (via <https://github.com/JoshuaKGoldberg/linting-typescript-in-2023/blob/main/.eslintrc.cjs>)
- Formatter: Prettier
